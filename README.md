## Other Information

This project is made using the tech stack below:
- Framework: Next JS
- Language: TypeScript
- Authentication: Next-Auth
- Styling: Styled-Component
- Testing: Jest
- Code Quality: ESLint
- Database: MongoDB
- Database ORM: Mongoose
- Content Management System (CMS): Sanity
